# react-native-circuit-ui

UI Kit for building React Native apps at Yonko Level

## Installation

```sh
npm install react-native-circuit-ui
```

## Usage


```js
import { multiply } from 'react-native-circuit-ui';

// ...

const result = await multiply(3, 7);
```
